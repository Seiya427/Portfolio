Programming Projects

Machine learning

K-Means Clustering Implementation:

In this project, I implemented the K-means clustering algorithm, which is commonly used in analysing patters in data. Using it, I was able to analyse very large data sets, including a series of transactions letting me find out about shopping habits across various countries. Commercially, K-means is used in similar scenarios, using data collected from the user to find out what kind of person they are. This was a very fun project that has taught me lots about unsupervised learning

Generative Pre-trained Transformer:

This project involved developing a GPT model to generate natural language and answer prompts. This was quite a fun task, and I used it to strengthen my python skills, and learn about how to use pyTorch, a commonly used python module for machine learning. I've learned lots about how a neural network works, and how we can train them to fulfill a task as well as possible.

Image Recognition System:

In this project, I developed an image recognition system using convolutional neural networks (CNNs). I used this to differentiate between animals, and also to identify numbers and letters from handwriting. This again served as an opportunity to learn lots about machine learning, and different neural network architectures, and I also was inspired to look more into the maths behind machine learning.

Other projects

Graphical calculator:

In this project, I made a GUI calculator which was able to manipulate and graph functions. Inspired by the Geogebra graphical calculator, this project also came with other personalised features such as function inversion, integration and differentiation.

Youtube video to mp3 converter:

I have to admit, this project is technically illegal, but it was really fun to make and is quite useful. The project provides a user interface of youtube and enables the user to search for videos, but instead of watching the videos, it enables you to download the sound or video file to watch / listen to offline.

French verb conjugation:

This project was made in order to helpme revise for my French GCSE, and was quite useful. It enables the user to either input a verb in french and conjugate it to a tense, mood and person of choice, or be tested on how to conjugate the words.

RSA Encryption:

In this project, I implemented the RSA cryptography algorithm, which is used in our daily lives to protect our data on the internet. In addition, I implemented shor's algorithm, a quantum computing algorithm that enables you to crack RSA. This was useful as it enabled me to learn about quantum computing, and how to use APIs in order to gain access to a quantum computer.
